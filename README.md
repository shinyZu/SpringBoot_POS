# Enterprise Level Application with Spring Boot

## POS system 
- Back-end & Front-end on seperate origins
- handled cross-origin requests (CORS issue)
- applied Layered Architecture
- easy dependency management with Starters 

## Used :

- Spring Boot
- Maven
- AJAX
- MySQL
- HTML
- CSS
- JavaScript
- JQuery
- Bootstrap
- SweetAlerts
- Toastr
